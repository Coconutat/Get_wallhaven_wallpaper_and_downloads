# 这是什么
我的期末考试的素材。代码可用，已经成功。  
爬取[wallhaven](https://wallhaven.cc/)网页的图片。
>特性
+ 1.可以分析wallhaven成列展示的图片，例如你进入了一个分类是：自然的页面。
+ 2.批量下载（**仅支持单线程**）
+ 更新
+ 3.增加All_in_one文件，直接启动即可联动两个模块。
